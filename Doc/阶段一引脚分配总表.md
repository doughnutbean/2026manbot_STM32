# Manbot 阶段一引脚分配总表

> 项目：`manbot`  
> 板卡：野火 STM32F103VE 指南者  
> 版本：阶段一验收版

## 1) 全组公共占用（禁止复用）

| 引脚    | 功能        | 使用人/模块  | 说明              |
| ----- | --------- | ------- | --------------- |
| `PB0` | 心跳LED     | D（系统集成） | 每500ms翻转，系统活性指示 |
| `PA9` | USART1_TX | D（系统集成） | 串口调试日志输出        |

## 2) 分工引脚分配（建议）（组内想要进行其他引脚占用可以沟通）

| 同学  | 模块职责      | 主用引脚                                              | 备用引脚               | 备注                  |
| --- | --------- | ------------------------------------------------- | ------------------ | ------------------- |
| A   | 语音输入/指令输入 | `PA2`(USART2_TX), `PA3`(USART2_RX)                | `PA0`, `PA1`       | 推荐接语音模块/UART模块      |
| B   | 动作控制/舵机   | `PA6`(TIM3_CH1), `PA7`(TIM3_CH2), `PB1`(TIM3_CH4) | `PA8`(TIM1_CH1)    | `PB0`不可用（已占用）       |
| C   | 显示模块      | `PB6`(I2C1_SCL), `PB7`(I2C1_SDA)                  | `PA5`/`PA4`(SPI备选) | SPI方案需与B协调 `PA7` 冲突 |
| D   | 系统集成/调度   | `PB0`(LED), `PA9`(日志)                             | `PC13`, `PB12`     | 负责全局配置与冲突仲裁         |

## 3) 冲突与约束（必须遵守）

| 冲突点                        | 风险            | 处理策略            |
| -------------------------- | ------------- | --------------- |
| `PA7`（B的PWM vs C的SPI_MOSI） | 模块互相干扰/无法同时工作 | 二选一：动作优先或显示改I2C |
| `PA9`（日志口）                 | 调试链路中断        | 阶段一固定给D，不外借     |
| `PB0`（心跳）                  | 系统运行状态不可视     | 阶段一固定给D，不外借     |

## 4) 变更流程（有更新需要记录）

暂无

## 5) 当前结论

- 阶段一“框架可运行”引脚分配已稳定。  
- 后续若新增外设，优先使用未占用口，避免改动 `PB0` / `PA9`。
